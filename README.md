🐍 Python Tkinter Dashboards Collection

This repository contains multiple GUI dashboard applications developed using Python and Tkinter. Each dashboard demonstrates different UI designs, interactive features, and event-driven programming concepts.

📌 Projects Included
1️⃣ Personal Profile Dashboard

A profile management dashboard that displays personal details, skills, and includes a dark mode toggle feature.

2️⃣ Python Practice Tracker

A task tracking system that allows users to add practice topics, mark them as completed, and monitor progress using a counter and progress bar.

3️⃣ Productivity Dashboard

Includes task monitoring, dynamic UI updates, mood tracking, and basic productivity tools.

4️⃣ Calculator Dashboard

A simple arithmetic calculator with radio button-based operation selection and dynamic result display.

🛠 Technologies Used

Python 3

Tkinter

ttk (Progress Bar)

Event-driven programming

GUI layout management (pack, grid, frames)

🎯 Key Concepts Demonstrated

GUI development

Event handling

State management

Progress tracking logic

These projects were built as part of Python GUI practice to strengthen understanding of interactive application development and improve logical implementation skills.
